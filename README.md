# Advent of JS 2024 Challenge Solutions

Solutions for the daily JS challenges from Advent of JS 2024 by Selfteach.me.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- No frameworks or libraries used

## Installation

```bash
git clone [repository-url]
cd AdventOfJs24
```

Open any challenge folder in your browser to view the solution.
